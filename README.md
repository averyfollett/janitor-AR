# janitor-AR
An AR app to assist with sanitization of commonly touched surfaces

# Documentation

**Engine used:** Unreal Engine 4.25.0

**Code/Language:** Currently blueprints-only derived from AR template

**Build tools:**
* android-28 SDK
* build-tools 28.0.3
* android-21 NDK
* JDK 11.0.7

**Plugins:**
* ARCore
* ARKit
